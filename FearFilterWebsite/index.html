What is Fear Filter?

Fear Filter is a chrome extension that checks every single image that is displayed on your screen and utilizes Microsoft's Computer Vision API - v1.0 to determine if the image triggers a user's phobia. For example, a person that has an extreme fear of spiders (arachnophobia), would set up a profile listing their fear and Fear Filter will "censor" every image containing spiders.

How does it work?

Fear Filter parses every visited website's HTML code for all image tags and then utilizes Microsoft's Computer Vision API to determine a list of tags for the images. It then cross checks those tags with the user's list of phobias and blocks out all of the content that may resemble the phobia. The blocked image will be replaced with a description of why the image was blocked and have a button to allow the user to view the content anyways.
